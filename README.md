# sHover 感应鼠标进入方向图片展示
一个简单的原生JavaScript小组件，它可以让你的展示图片感应鼠标进入方向，从而让悬浮层做出不同的运动效果。
多个图片放在一起效果更酷炫噢！它的使用和设置都非常方便。
而且兼容性上表现非常出色，在IE5及以上都可能完美运行。
不需要JQuery哦。
